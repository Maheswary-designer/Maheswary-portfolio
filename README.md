# Maheswary Vangaru — Droow Inspired Dark Portfolio

This is a customized dark portfolio website built with HTML, CSS and JavaScript.

## Included files

- `index.html` — main portfolio page
- `css/styles.css` — responsive dark portfolio styling
- `js/main.js` — preloader, slider, side menu, scroll reveal, cursor and smooth interactions
- `assets/resume/MAHESWARY-VANGARU-Resume.pdf` — resume download file
- `assets/images/` — Live Projects section preview images

## How to use on GitHub Pages

1. Upload all files to a GitHub repository.
2. Keep `index.html` in the root folder.
3. Go to repository Settings → Pages.
4. Select branch `main` and folder `/root`.
5. Save and open the GitHub Pages live link.

## How to edit

- Change text in `index.html`.
- Change colors and spacing in `css/styles.css`.
- Change slider and interaction behavior in `js/main.js`.
- Replace online image URLs with your own images if needed.
- Edit Live Projects cards and links inside the `#live-projects` section in `index.html`.
